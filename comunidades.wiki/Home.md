# Administración de comunidades
## Requisitos técnicos
* PHP 8.1+
* Laravel 9.7+
* Jetstream con livewire, exclusivamente para autenticación
* Bootstrap 4.6+
* Plantilla AdminLTE 3.1+
* SGBD PostgreSQL 12.0+
